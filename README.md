# Turtle Rested XP

A small draggable rested XP progress bar for Turtle WoW.

<img width="275" height="59" alt="image" src="https://github.com/user-attachments/assets/65e091af-0411-4cff-bd29-302ea509653a" />

## Features

- Shows a progress bar with your current rested XP percentage
- **Auto-shows** when you enter a resting zone (inn or city) (configurable)
- **Auto-hides** when you leave a resting zone (configurable)
- Draggable — position is saved between sessions
- Close button to dismiss the bar until the next resting session
- Hover tooltip with exact percentage and XP pool size
- Color shifts from blue to green as rested XP fills up

## Commands

- `/restedxp` — Show the options pane
- `/restedxp show/toggle` — Show/hide the rested XP bar manually
- `/restedxp reset` — Reset the position of the frame
